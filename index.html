Por onde eu começo ?

1° - Instalar o Nodejs

Instalando a CLI do Grunt

npm install -g grunt-cli

Iniciar o seu projeto

npm init //na pasta do projeto hein!!

Instalar o Grunt na pasta do seu projeto

npm install grunt --save-dev

Baixar o plugin desejado

npm install grunt-contrib-sass --save-dev //sass

Carregar o plugin desejado

grunt.loadNpmTasks('grunt-contrib-sass'); //após o grunt.initConfig

Configurar o plugin

Cada plugin tem a sua forma única de ser configurado que é informado na própria documentação
